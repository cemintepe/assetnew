# Asset Mobil – Dokümantasyon İndeksi

Bu klasör, Asset Mobil projesinin teknik ve kavramsal dokümantasyonunu içerir. Başka bir geliştirici veya yapay zekaya projeyi devralırken **bu dosyaları sırayla okuması** önerilir.

---

## Okuma Sırası

1. **[PROJE_GENEL_BAKIS.md](./PROJE_GENEL_BAKIS.md)** – Proje tanımı, kullanıcı rolleri, iş akışları, ekranlar
2. **[TEKNIK_MIMARI.md](./TEKNIK_MIMARI.md)** – Teknoloji yığını, dosya yapısı, mimari kararlar
3. **[VERI_AKISI_KAVRAMLAR.md](./VERI_AKISI_KAVRAMLAR.md)** – SQLite şeması, state yapısı, prop geçişleri, veri akışları
4. **[API_REFERANSI.md](./API_REFERANSI.md)** – REST API endpoint’leri, request/response formatları
5. **[GELECEK_DEVAM.md](./GELECEK_DEVAM.md)** – Tamamlanan/planlanan özellikler, teknik borç, devam rehberi
