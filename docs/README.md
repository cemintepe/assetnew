# ASSETSUP / Asset Mobil – Dokümantasyon İndeksi

Bu klasör, Asset Mobil (ASSETSUP) projesinin teknik ve kavramsal dokümantasyonunu içerir. Başka bir geliştirici veya yapay zekaya projeyi devralırken **bu dosyaları sırayla okuması** önerilir.

---

## Okuma Sırası

1. **[PROJE_GENEL_BAKIS.md](./PROJE_GENEL_BAKIS.md)** – Proje tanımı, kullanıcı rolleri, iş akışları, ekranlar
2. **[TEKNIK_MIMARI.md](./TEKNIK_MIMARI.md)** – Teknoloji yığını, dosya yapısı, mimari kararlar (Supabase + REST API + SQLite)
3. **[VERI_AKISI_KAVRAMLAR.md](./VERI_AKISI_KAVRAMLAR.md)** – SQLite şeması, context/state yapısı, prop geçişleri, veri akışları
4. **[API_REFERANSI.md](./API_REFERANSI.md)** – Supabase tablo yapıları + REST API endpoint’leri, request/response formatları
5. **[GELECEK_DEVAM.md](./GELECEK_DEVAM.md)** – Tamamlanan/planlanan özellikler, teknik borç, devam rehberi

---

## Hızlı Referans

| Konu | Doküman |
|------|---------|
| ST/T akışları nasıl çalışıyor? | PROJE_GENEL_BAKIS.md |
| Hangi teknolojiler ve servisler kullanılıyor (Supabase, REST, SQLite)? | TEKNIK_MIMARI.md |
| Context/state ve props nasıl organize? | VERI_AKISI_KAVRAMLAR.md |
| Hangi Supabase tabloları / REST endpoint’leri var? | API_REFERANSI.md |
| Ne yapıldı, ne kaldı, dikkat edilecekler? | GELECEK_DEVAM.md |
